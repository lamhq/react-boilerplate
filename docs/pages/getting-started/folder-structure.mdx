The most important files and directories in the root directory are:

- `index.html`: the main HTML file that acts as the [entry point](https://vitejs.dev/guide/#index-html-and-project-root) for your application.
- `vite.config.ts`: [configuration file](https://vitejs.dev/config/#configuring-vite) for Vite project.
- `.github/`: contain Github Workflow configuration files and pull request template for this project.
- `.vscode/`: contain VS Code's setting and launch configuration to run & debug the project.
